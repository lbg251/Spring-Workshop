## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g., `spai-workshop-2026`)
2. Upload both `index.html` and `poi-logo-dark.png`
3. Go to **Settings** → **Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose `main` branch and `/ (root)` folder
6. Click **Save**

Your site will be live at `https://[username].github.io/[repo-name]/`

## Custom Domain (Optional)

To use a custom domain like `spai2026.pibbss.ai`:

1. Create a file called `CNAME` with just the domain name
2. Add a CNAME record in your DNS pointing to `[username].github.io`
